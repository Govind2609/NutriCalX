# Nutritional Information Calculator

A web application that provides detailed nutritional information for various food items. Users can search for any food and get comprehensive nutrition facts including calories, macronutrients, vitamins, and minerals for a specified quantity.

## Features

- Search for any food item
- View detailed nutritional information per 100g or custom quantity
- See food images when available
- Responsive design that works on all devices
- No API key required (uses Open Food Facts API)

## Technologies Used

- Frontend: HTML5, CSS3, JavaScript (Vanilla JS)
- Backend: PHP
- API: Open Food Facts (https://world.openfoodfacts.org/)

## Setup

1. Clone the repository
2. Set up a local PHP server (e.g., XAMPP, WAMP, or `php -S localhost:8000`)
3. Open `index.html` in your web browser

## Usage

1. Enter a food item (e.g., "apple", "banana", "chicken breast")
2. Specify the quantity in grams
3. Click "Calculate Nutrition Info" to view detailed nutritional data

## Note

This application uses the Open Food Facts database, which is community-driven. Some less common foods might not be available or have incomplete information.
